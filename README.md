# gulp-front-end-template
Front-end template for Gulp

## What it can do?
- LESS
- CSS autoprefixer
- CSS, JS, image minify
- Sprites
- SVG sprites

## Getting Started
1. Copy source, lib, bower.json, package.json, gulpfile.js.
2. Run
	$ npm install
3. Run 
	$ bower install
4. Start server - run 
	$ gulp
5. Open http://localhost:9999/
